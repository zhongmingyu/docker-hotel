环境：能连上国际互联网的windows11电脑。

1，访问 https://www.docker.com/products/docker-desktop/  下载并安装Docker Desktop Installer.exe。重启电脑。
2，运行安装好的Docker Desktop。Continue without signing in->skip。
3，鼠标右键选择“在终端中打开”
4，输入 'run.bat'  回车。
===============================================================================
Environment: A Windows 11 computer with access to the international internet.

1，Visit https://www.docker.com/products/docker-desktop/ to download and install Docker Desktop Installer.exe. Restart the computer.
2，Run the installed Docker Desktop. Choose "Continue without signing in" -> "Skip".
3，Right-click and select "Open in Terminal".
4，Enter 'run.bat' and press Enter.
===============================================================================
Môi trường: Máy tính Windows 11 có thể kết nối với Internet quốc tế.

1、Truy cập https://www.docker.com/products/docker-desktop/ để tải và cài đặt Docker Desktop Installer.exe. Khởi động lại máy tính.
2、Chạy Docker Desktop đã cài đặt. Chọn "Continue without signing in" -> "Skip".
3、Nhấn chuột phải và chọn "Open in Terminal".
4、Nhập lệnh 'run.bat' và nhấn Enter.
