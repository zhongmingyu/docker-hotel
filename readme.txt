1，访问 https://www.docker.com/products/docker-desktop/  下载并安装Docker Desktop Installer.exe。重启电脑。
2，运行安装好的Docker Desktop。Continue without signing in->skip。启动Docker Engine。如果启动不了，则运行 updateWsl.bat。
3，输入 'run.bat'  回车。
